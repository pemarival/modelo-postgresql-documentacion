# Título
Implementación de un mecanismo de auditoría para registrar operaciones críticas del sistema.

# Contexto
El sistema gestiona procesos sensibles como:
- reservas de vuelos
- emisión de tickets
- pagos y facturación
- check-in de pasajeros
- gestión de usuarios

Estas operaciones pueden ser ejecutadas por distintos usuarios del sistema y requieren trazabilidad para fines de seguridad, control y diagnóstico.

En sistemas reales de aerolíneas es necesario poder identificar:
- quién realizó una operación
- cuándo se realizó
- qué información fue modificada

# Problema
Actualmente el sistema no dispone de un mecanismo centralizado para registrar las acciones realizadas sobre las entidades críticas del sistema.

Esto puede generar dificultades para:
- auditar cambios realizados por usuarios
- detectar errores operativos
- investigar incidentes o fraudes
- analizar el comportamiento del sistema

# Decisión
Se decidió implementar un mecanismo de auditoría de operaciones que registre automáticamente las acciones realizadas sobre las entidades críticas del sistema.

Las operaciones que serán auditadas incluyen:
- creación de reservas
- emisión de tickets
- registro de pagos
- modificación de vuelos
- cambios en roles o permisos de usuarios

La auditoría registrará información como:
- usuario que ejecutó la acción
- tipo de operación realizada
- entidad afectada
- fecha y hora del evento

# Justificación técnica
La implementación de auditoría permite:
- mejorar la trazabilidad del sistema
- detectar errores o comportamientos anómalos
- cumplir buenas prácticas de seguridad en sistemas empresariales

Además, este mecanismo puede integrarse con herramientas de monitoreo y registro de eventos.

# Consecuencias o impacto esperado
### Impactos positivos
- mayor trazabilidad de operaciones del sistema
- facilidad para investigar incidentes
- mejor control sobre acciones realizadas por usuarios
  
### Impactos negativos
- incremento en el volumen de registros de eventos
- necesidad de administrar almacenamiento de logs a largo plazo
