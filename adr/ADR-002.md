# Título
Implementación de control de acceso basado en roles (RBAC).

# Contexto
El sistema incluye múltiples tipos de usuarios:
- Administradores del sistema
- Personal de aeropuerto
- Agentes de ventas
- Personal de check-in
- Personal de abordaje
- Clientes

Cada uno de estos actores requiere distintos niveles de acceso a la información y funcionalidades del sistema.

El modelo de datos ya incluye las tablas:
- security_role
- security_permission
- user_role
- role_permission
- user_account

Esto permite implementar un esquema de control de acceso robusto.

# Problema
Sin un sistema claro de control de acceso basado en roles pueden ocurrir los siguientes problemas:
- Usuarios con acceso excesivo a información sensible.
- Falta de control sobre quién puede ejecutar operaciones críticas.
- Dificultad para mantener la seguridad a medida que el sistema crece.
  
# Decisión
Se adopta un modelo de Control de Acceso Basado en Roles (RBAC).

La estructura de autorización queda definida de la siguiente forma:
Usuario → Rol → Permiso

- Administrador: gestión completa del sistema 
- Agente de ventas: reservas, emisión de tickets
- Agente de check-in: check-in y asignación de asiento
- Agente de abordaje: validación de boarding pass
- Cliente: consulta de reservas

# Justificación técnica
El modelo RBAC ofrece varias ventajas:

- Escalabilidad en la gestión de accesos.
- Simplificación del mantenimiento de permisos.
- Separación entre identidad, rol y autorización.
- Compatibilidad con estándares de seguridad empresarial.

Además, el modelo ya está soportado en el esquema mediante:
- user_role
- role_permission

lo que evita rediseños estructurales.

# Consecuencias o impacto esperado
### Positivo 
- Mayor control sobre el acceso a funcionalidades.
- Reducción del riesgo de accesos indebidos.
- Mejora en la trazabilidad de operaciones por usuario.
  
### Negativo
- Necesidad de mantener catálogo de roles y permisos.
- Mayor complejidad inicial en la configuración.
